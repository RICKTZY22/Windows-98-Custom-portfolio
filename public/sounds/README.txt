CUSTOM SYSTEM SOUNDS
====================

Drop audio files into this folder to replace the built-in synthesized sounds.
The OS checks for these exact file names (mp3, wav, or ogg extension):

  startup.mp3      played when Windows reaches the desktop
  shutdown.mp3     played when shutting down
  error.mp3        error message boxes + crash beeps
  warn.mp3         warning message boxes
  click.mp3        UI click
  menuOpen.mp3     opening menus
  recycle.mp3      sending files to the Recycle Bin
  networkUp.mp3    ethernet connected
  networkDown.mp3  ethernet disconnected
  launch.mp3       opening a program window
  minimize.mp3     minimizing a window
  restore.mp3      restoring a window
  ding.mp3         general notification
  tada.mp3         success fanfare

Example: put the real Windows 98 startup sound here as "startup.mp3" and it
plays instead of the synth chord. Files you do not provide fall back to the
built-in synthesized versions automatically.

Note: browsers only allow audio after your first click/keypress on the page,
so the very first boot of a fresh visit may be silent until you interact.
